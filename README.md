## ブルアカのカフェタッチタイマー

### 機能

1. 3時間おきにタイマーを設定します
2. タイマーが終了したらアプリ画面が点滅します
3. 4時と16時のカフェ更新タイミングでも同様に点滅します
4. 3時間以内にカフェ更新がある場合はタイマーが動作せずに警告が出ます
